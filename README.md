# AI-Website

Современный веб-сайт, созданный с помощью ИИ и быстрых прототипов.

**Технологии:**
- Next.js (React)
- TypeScript
- Tailwind CSS
- ИИ-инструменты (Claude, Grok, ChatGPT)
- 3D-элементы (Tripo Studio, Three.js)

---

### 🎯 О проекте

Это демонстрационный проект, показывающий возможности **Vibe Coding** — быстрого создания веб-приложений с помощью искусственного интеллекта.

### ✨ Основные возможности

- Интерактивный 3D-интерфейс
- Генерация контента с помощью ИИ
- Адаптивный дизайн
- Быстрая загрузка и современный UX

### 🚀 Как запустить локально

```bash
# Клонировать репозиторий
git clone https://github.com/8VasilevS8/AI-Website.git

# Перейти в папку
cd AI-Website

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
